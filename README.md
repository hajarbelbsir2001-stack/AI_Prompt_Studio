# AI Prompt Studio

## Description

AI Prompt Studio est une application développée avec Python, FastAPI et Streamlit permettant de générer automatiquement des prompts intelligents et structurés selon les besoins de l'utilisateur.

## Fonctionnalités

- Génération automatique de prompts
- Interface graphique moderne avec Streamlit
- Historique des prompts générés
- Suppression de l'historique
- Téléchargement au format TXT
- Téléchargement au format PDF
- Statistiques des prompts générés
- Interface responsive et intuitive

## Technologies utilisées

- Python
- FastAPI
- Streamlit
- ReportLab
- Requests
